интерфейс: Swing
База Данных: MS SQL Server 2014
взаимодействие с БД: JDBC

Эта программа - первая версия системы по асинхронному запуску заданий.
Задачами являются запросы к базе данных MS SQL Server 2014.

В папке с исполняемым файлом sqlmeter-1.jar должны находиться конфигурационный файл config.ini и подпапка lib в которой находится драйвер к Базе данных sqljdbc-4.jar . Другие посторонние библиотеки не использовались. В конфигурационном файле config.ini прописывается доступ к Базе Данных. При необходимости эти настройки могут быть изменены.

Приложение работает с двумя таблицами (таблица Задачи и таблица Расписание), которые расположены в двух вкладках. Записи в таблицах редактируются с помощью кнопок Добавить, Изменить, Удалить.

Таблица Расписание имеет дополнительную кнопку Запустить, при нажатии на которую выполняется Задача (запрос к БД). В карточке расписания есть поле параметры, которое на данный момент может обрабатывать два варианта параметров, записанных в формате XML:

1) Настройки подключения к БД, типа:
<connection host="localhost" port="1433" dbname="schedule" username="user" pass="12345"> </connection>
Если в параметрах указаны настройки подключения, то соединение с БД будет устанавливаться используя эти настройки. Если этот параметр отсутствует соединение происходит по настройкам из файла config.ini.

2)Уточняющий параметр сортировки, типа:
<orderby column="schd_ID"></orderby>
В этом параметре можно указать поле по которому будет происходить сортировка по возрастанию. К SQL запросу будет прибавлено ORDER BY с соответствующим полем.

После запуска задачи в окне отображается результат работы. А после завершения запроса, в поле "Время выполнения" записывается время за которое был получен ответ.

Копия базы находится в файле schedule.bak
